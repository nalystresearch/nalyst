# Security Policy

We appreciate responsible disclosure of any security issues. Please do not open a public issue for vulnerabilities.

## Supported versions

Nalyst is currently in pre-1.0 development. Security fixes will be released on the latest minor version when applicable.

## Reporting a vulnerability

- Email: dev@nalyst.ai
- Include as much detail as possible (version, environment, reproduction steps, potential impact).
- We aim to acknowledge reports within 5 business days and will coordinate a fix and disclosure plan.
