# Support

Need help with Nalyst? Start here.

## How to get help

- Check existing documentation in `doc/` and runnable examples in `examples/`.
- Search past issues or discussions (when the public tracker is available).
- If you find a bug, open an issue with a minimal reproduction (dataset shape, code, expected vs. actual behavior).

## Response expectations

Nalyst is actively developed; responses are typically within a few business days. Security issues should follow SECURITY.md.
